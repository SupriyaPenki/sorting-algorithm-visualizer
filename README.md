# Sorting Algorithm Visualizer

This project is an interactive web application that helps users understand how sorting algorithms work through visualization.

## Algorithms Implemented

- Bubble Sort
- Merge Sort
- Quick Sort

## Features

- Interactive sorting visualization
- Random array generation
- Adjustable array size
- Sorting speed control
- Java implementation of algorithms
- Time and space complexity display

## Technologies Used

- HTML
- CSS
- JavaScript
